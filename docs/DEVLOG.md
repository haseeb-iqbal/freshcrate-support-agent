## Motivation

Up until this point, my use of AI has been limited to identifying and solving bugs, writing and optimising functions, and generating tests. Though I have used AI to edit multiple files for specific use cases in the past, I had not used it to generate a multitude of files based on requirements, let alone create an entire app. This project is my personal experiment through which I can take my AI-assisted development skills to the next level.

## Decisions

A RAG chatbot isn't the most novel idea, but it's interesting and complex enough to develop as an experimental project. A project like a SaaS portfolio website would have been too focused on designing and creating components, whereas I want to build actual features.

I decided to use Claude for AI-assisted development because it seems to be well-reviewed and popular. Local LLMs such as GLM 5.2 are another option, but they need both decent hardware and context to work correctly, both of which would take considerable time to set up.

## Costs

I decided to get a Claude Pro subscription at $20 a month, which is a little more expensive than my discounted ChatGPT + Copilot subscriptions, which I then cancelled. I have only hit my daily max token limit once so far, but I expect I will hit it more often going forward and may need to upgrade my Claude subscription.

I also bought $5 worth of OpenAI API credits for the RAG chatbot. This was the minimum amount I could put in the wallet, and it should last a long time if used just for the current project.

Deployment and database hosting are free via Vercel and Supabase.
